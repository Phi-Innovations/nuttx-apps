# Hellocpp

This is a HelloWorld project done in C++ running on Nuttx.

# Features

* Use Nuttx as a library (nuttx-export-9.1.0)
* Use Cmake as build engine
* Has automation to load firmware on the board using Segger J-Link

# Build instruction

```
$ make default
```

# Load the firmware

```
$ make run
```
